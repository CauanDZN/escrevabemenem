# Footer

The `Footer` component is used to display a footer section at the top of the page.

<Canvas of={FooterStories.Default} />

<Controls />

## Usage

```jsx
import Footer from './Footer';

<Footer />
```

## Props

The `Footer` component does not accept any props.

## Styling

The `Footer` component uses Tailwind CSS for styling.

You can customize the styling of the `Footer` component by modifying the Tailwind CSS classes applied to the footer element. The component uses responsive classes to adjust the layout based on screen size.
